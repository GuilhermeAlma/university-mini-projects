# 🎓 Universidade de Aveiro : Academic Projects Archive

Welcome to my academic repository! This space serves as a centralized archive for coursework, lab assignments, and data analysis projects completed during my degree at the [Universidade de Aveiro Physics Bachelors](https://www.ua.pt/). 

While my larger, standalone projects have their own dedicated repositories, this collection highlights my day-to-day coding progression, problem-solving exercises, and foundational learning across various subjects.

To make this repository accessible globally, all code documentation and `README.md` summaries are written in English. However, to preserve the authenticity of the original academic submissions, some of the attached PDF reports and presentation slides remain in their native Portuguese.

## 📂 Repository Structure

The projects are organized by subject area. Inside each folder, you will find a dedicated `README.md` with specific details about that particular assignment, the tech stack used, and instructions on how to run it.

* **`python_geochemical_analysis/`**
  A Command-Line Interface (CLI) application built in Python that processes geochemical data to map rock types and compute optimized geographical sampling routes using distance algorithms.

* **`matlab_gui_application/`**
  An interactive educational game and physics simulator built using MATLAB App Designer. It models 2D projectile kinematics, allowing users to launch a projectile and hit targets across different planetary gravity environments.

* **`python_exoplanets_data_analysis/`**
  A data exploration project using Python (Pandas, Matplotlib) to uncover trends regarding planetary habitability, mass, radius, and discovery methods over time.

* **`r_medical_analysis/`**
  A biostatistics project utilizing R to perform linear and logistic regression on a medical dataset, aiming to identify which clinical biomarkers are the strongest predictors of heart attacks.
